# Veilium Browser

**Veilium** is a desktop browser designed with privacy and speed at its core. It gives you a clean, modern browsing experience while making sure your screen content stays protected from unwanted captures — even at the operating system level.

### What Makes Veilium Different

- **Built-in Screen Protection** — Your browsing stays private, invisible to screen capture tools
- **Smart Memory Management** — Tabs you're not using are automatically put to sleep so your computer stays fast
- **Clean, Modern Interface** — A simple and intuitive design that gets out of your way
- **Intelligent Search Bar** — One bar for everything — type a website address or search the web, it figures out what you mean
- **Browsing History** — A dedicated, privacy-focused history page to find what you visited before
